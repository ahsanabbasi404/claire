# Fonts Directory

This directory should contain the following font files:

## Required Fonts:

1. **CooperLtBT-Regular** (for main headings)
   - `CooperLtBT-Regular.woff2` (preferred)
   - `CooperLtBT-Regular.woff` (fallback)
   - `CooperLtBT-Regular.ttf` (fallback)

2. **Saans_TRIAL-Regular** (for body text)
   - `Saans_TRIAL-Regular.woff2` (preferred)
   - `Saans_TRIAL-Regular.woff` (fallback)
   - `Saans_TRIAL-Regular.ttf` (fallback)

3. **Saans_TRIAL-Medium** (for medium weight text)
   - `Saans_TRIAL-Medium.woff2` (preferred)
   - `Saans_TRIAL-Medium.woff` (fallback)
   - `Saans_TRIAL-Medium.ttf` (fallback)

## How to Add Fonts:

1. Download the font files from your font provider
2. Convert to web formats (.woff2, .woff) if needed using tools like:
   - https://cloudconvert.com/
   - https://www.fontsquirrel.com/tools/webfont-generator
3. Place the font files in this directory
4. The fonts will automatically load via the @font-face declarations in `app/globals.css`

## Font Sources:

- **CooperLtBT-Regular**: Available from Bitstream or font vendors
- **Saans_TRIAL**: Available from xFonts.pro or other font distributors

Note: Ensure you have proper licensing for commercial use of these fonts.
